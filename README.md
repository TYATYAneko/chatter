# StudyBoard

グループ学習ノート共有アプリケーション

## バージョン

**v1.1.0**

## 概要

StudyBoardは、グループでノートを共有できるWebアプリケーションです。個人情報不要で手軽に利用できます。

## 機能

### 認証
- **アクセスコード**: 10桁の数字でサイトにアクセス
- **アカウント作成**: ニックネームとパスワードのみ（メールアドレス不要）
- **ログイン/ログアウト**

### グループ
- **グループ作成**: 任意の名前でグループを作成
- **招待コード**: 6桁の英数字（自動生成または自分で設定可能）
- **グループ参加**: コードを入力して参加
- **グループ退出**: メニューから退出可能
- **グループ情報**: メンバー一覧などを確認

### ノート共有
- **メモ投稿**: テキストメモの投稿・共有
- **リアルタイム更新**: 1秒ごとに自動更新
- **システム通知**: 参加・退出の通知

### 設定
- **テーマ**: ライト / ダーク / ブルー / グリーン
- **フォントサイズ**: 小 / 中 / 大
- 設定は自動保存され、次回も反映

## 使い方

1. `index.html` をブラウザで開く
2. アクセスコードを入力（初期値: `1234567890`）
3. アカウントを作成またはログイン
4. グループを作成、またはコードを入力して参加
5. ノート共有開始

## アクセスコードの変更

`app.js` の5行目を編集:

```javascript
const SITE_PASSWORD = '1234567890';  // ここを変更
```

## 技術仕様

- **フロントエンドのみ**: サーバー不要
- **データ保存**: localStorage / sessionStorage
- **同期**: 同一ブラウザの別タブ間で同期可能

## 注意事項

- 異なる端末間での共有にはサーバーが必要です
- データはブラウザに保存されるため、ブラウザのデータを削除すると消えます
