# Chatter

シンプルなチャットルームアプリケーション

## バージョン

**v1.0.0**

## 概要

Chatterは、個人情報不要で手軽にチャットルームを作成・参加できるWebアプリケーションです。

## 機能

### 認証
- **サイトパスワード**: 10桁の数字でサイトにアクセス
- **アカウント作成**: ユーザー名とパスワードのみ（メールアドレス不要）
- **ログイン/ログアウト**

### ルーム
- **ルーム作成**: 任意の名前でルームを作成
- **ルームコード**: 6桁の英数字（自動生成または自分で設定可能）
- **ルーム入室**: コードを入力して参加
- **ルーム退出**: メニューから退出可能
- **ルーム情報**: メンバー一覧などを確認

### チャット
- **メッセージ送信**: テキストメッセージの送受信
- **リアルタイム更新**: 1秒ごとに自動更新
- **システムメッセージ**: 入室・退出の通知

### 設定
- **テーマ**: ライト / ダーク / ブルー / グリーン
- **フォントサイズ**: 小 / 中 / 大
- 設定は自動保存され、次回も反映

## 使い方

1. `index.html` をブラウザで開く
2. サイトパスワードを入力（初期値: `1234567890`）
3. アカウントを作成またはログイン
4. ルームを作成、またはコードを入力して入室
5. チャット開始

## サイトパスワードの変更

`app.js` の5行目を編集:

```javascript
const SITE_PASSWORD = '1234567890';  // ここを変更
```

## 技術仕様

- **フロントエンドのみ**: サーバー不要
- **データ保存**: localStorage / sessionStorage
- **同期**: 同一ブラウザの別タブ間で同期可能

## 注意事項

- 異なる端末間でのチャットにはサーバーが必要です
- データはブラウザに保存されるため、ブラウザのデータを削除すると消えます
